Varför ska jag lära mig programmering?

Det finns tusen olika anledningar till varför du ska lära sig programmering. Först och främst är det kul och superkreativt! Du får lösa problem och får möjlighet att skapa produkter som du tidigare bara haft idéer om, tex en hemsida, app eller blogg.


Utvecklare är idag också eftertraktade på arbetsmarknaden och att kunna programmera ger dig ett tydligt försprång. Utöver det finns det stora könsskillnader inom programmeringsvärlden. Vi tycker därför att det är grymt viktigt att alla tjejer som vill testa på programmering får chansen att göra det. Vi vill jobba mot en mer jämnlik techbranch.


Att lära sig programmering är heller inte bara användbart om man vill jobba som programmerare eftersom programmering används i många olika jobb idag. Du kan egentligen jobba som vad som helst och ändå ha användning av att kunna programmering.


Vad är front-end och back-end programmering?

I programmeringsvärlden kanske du har hört talas om front-end och back-end. Detta är vad en applikation ofta består av. Dock kan man bygga statiska hemsidor i endast frontend, tex den här! Front end är den del som användaren ser och interagerar med. En front-end-utvecklare kan tex jobba i tekniker som HTML, CSS, JavaScript eller React. Det är Front-End som du lär dig på de flesta av våra kurser.

Back-end är det som händer under ytan, tex en databas som lagrar alla blogginlägg. Java, Python och SQL är exempel på språk som kan användas av en back-end-utvecklare.


Varför ska jag lära mig hos er?

För oss handlar det inte bara om programmering utan om att skapa ett säkert, tryggt och inspirerande space där alla kan känna sig välkomna. Våra event och kurser genomsyras av detta. För oss är det också viktigt att hålla igång våra communityn efter kursens slut. Att hoppa på en terminskurs eller intensivdag är bara början!

Vi försöker också hålla nere våra priser genom att samarbeta med olika sponsorer och på så vis göra det möjligt för fler att gå våra kurser. Utöver det är både vi, våra lärare och alla andra grymma personer som hjälper till otroligt taggade på att lära ut så mycket vi bara kan. Vi håller oss ständigt uppdaterade om nya teknologier och uppdaterar vad vi lär ut.

Vad är HTML?

HTML står för Hyper Text Markup Language och används för att strukturera en web-sida och dess innehåll.

• HyperText  är det som gör att du kan röra dig runt på nätet. När du klickar på viss text som kallas hyperlänkar tar den dig till en annan sida. Markup är det som informerar browsern om hur texten ska vara formatterad. Detta görs med HTML taggar som sitter runt texten.

• Man kan säga att HTML ger sidan dess bas-struktur, där man sedan kan lägga på CSS för att ändra utseendet.


Vad är CSS?

CSS står för Cascading Style Sheets. Den bestämmer hur HTML element ska se ut. Du kan till exempel använda CSS för att ändra typsnitt, färg eller storlek på hemsidans innehåll. Du kan också skapa t.ex animationer!


Vad är JavaScript?

JavaScript används för att programmera vad som ska hända på en hemsida. Det är det tredje lagret av web-teknologier efter HTML och CSS. Några grunläggande byggstenar av JavaScript är variabler, funktioner och metoder. Dessa koncept återkommer genom programmering generellt och du kommer se dem igen när du börjar utforska andra språk.

Det är ett språk som gör att du på din hemsida kan skapa dynamisk uppdaterande innehåll, göra sidorna interaktiva, animera bilder och mycket mer!
