About Code Excursion

âœ»

Code Excursion is a female coding community where women learn programming. The focus of Code Excursion is to learn in a helpful, passionate and including community with a bunch of cheerful, curious and supporting girls who start the journey of becoming a web developer together. At the moment we have courses and events in Stockholm. Our friends in Switzerland have similar events in Zurich.

Background

The seed to Code Excursion came the summer of 2017. We had been trying to learn coding for a long time without much success. We felt like we were missing a safe and inspiring space to learn. To fix this problem we created Code Excursion!

Now

At the moment we have a Front End developer courses at the wonderful Knackeriet in Stockholm. Furthermore, we hold workshops about code related topics which are open for more girls. Stay tuned

Future

In the spring of 2018 we will have new courses, worksops and events. If you are interested in joining, follow us and subscribe to our newsletter to make sure you get the information!
