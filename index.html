About Code Excursion

Code Excursion is a female coding community where women learn programming. The focus of Code Excursion is to learn in a helpful, passionate and including community with a bunch of cheerful, curious and supporting girls who start the journey of becoming a web developer together. At the moment we exist in Sweden and Switzerland.

Background

The seed to Code Excursion came the summer of 2017. We (Louise and Corina) had been trying to learn coding for a long time without much success. We both felt like we were missing a safe and inspiring space to learn. To fix this problem we created Code Excursion! Now Code Excursion Stockholm is organized by Louise and Caroline while Code Excursion Switzerland is organized by Corina and Celine

Now

Our first group of girls have graduated from our Front End developer course in Stockholm. This group will keep learning together as many want to further deepen their knowledge. We have also started with our second group who will learn together for 12 weeks. Furthermore, we hold workshops about code related topics which are open for more girls. Stay tuned

Future

In the fall of 2018 we will start a new group in Stockholm. If you are interested in joining, follow us and subscribe to our newsletter to make sure you get the information earlier than others! We are also planning to create our own digital course and a mentorship program.
